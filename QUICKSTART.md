# 快速启动指南

## 前置要求

- Python 3.8+
- Node.js 14+
- npm 或 yarn

## 快速开始

### 1. 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 启动后端服务

```bash
python start.py
```

后端服务将在 http://localhost:8000 启动

### 3. 安装前端依赖（新终端窗口）

```bash
cd frontend
npm install
```

### 4. 启动前端服务

```bash
npm start
```

前端将在 http://localhost:3000 启动

## 首次使用

1. 打开浏览器访问 http://localhost:3000
2. 点击"发现新矿机"按钮
3. 系统会自动扫描配置的IP范围（10.102.0.1-10.102.0.255 和 10.102.1.1-10.102.1.65）
4. 等待扫描完成，矿机会自动显示在列表中

## 功能说明

- **仪表板**：显示所有矿机的概览信息
- **矿机详情**：点击矿机卡片查看详细信息
  - 实时温度、功耗、算力等数据
  - 算力板详细信息
  - 历史趋势图表
  - 运行日志

## 常见问题

**Q: 无法发现矿机？**
A: 检查：
1. 矿机是否在线
2. IP地址是否在配置范围内
3. 防火墙是否允许4028端口
4. 网络连接是否正常

**Q: 后端启动失败？**
A: 检查：
1. Python版本是否为3.8+
2. 是否安装了所有依赖
3. 8000端口是否被占用

**Q: 前端无法连接后端？**
A: 检查：
1. 后端服务是否正常运行
2. 前端配置的API地址是否正确（默认 http://localhost:8000）
